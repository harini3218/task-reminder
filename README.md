# Task Reminder Application

A Spring Boot application to manage tasks and send email reminders when tasks are due.

---

## Features
- Create, update, delete tasks
- Email reminders using Spring Scheduler
- H2 in-memory database
- CSV export of tasks
- REST APIs using Spring Boot

---

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- Spring Mail
- H2 Database
- Maven

---

## Frontend

Simple web interface built using:
- HTML
- CSS
- JavaScript

Location:
frontend/index.html

To run:
Open index.html in browser after starting backend server.
